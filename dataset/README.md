# Berry Dataset

This folder represents the berry image dataset used in the repository.

The full dataset contains **2,519 images** across five berry classes:

- Tayberry: `500`
- Wineberry: `500`
- Blackberry: `580`
- Blueberry: `615`
- White Mulberry: `324`

## Class Structure

```text
dataset/
├── Tayberry/
├── Wineberry/
├── black berry/
├── blueberry/
└── white mulberry/
```

## Usage

The dataset is used for:

- berry image classification
- explainable AI analysis with Grad-CAM
- benchmarking attention-based deep learning models for fine-grained berry recognition
